# 0036 统一新手生成设置

## Status

Accepted

## Context

工作台的视频模式越来越多，用户在每个模式里都要分别理解时长、音频、字幕和导出内容，容易出现“没勾字幕却生成字幕”“混剪到底按素材还是按音频时长”“生成音频会不会进入本次任务”等误解。

## Decision

所有制作模式在资料区顶部显示统一的“生成前设置”，集中说明四件事：

- 视频时长依据：数字人/口型模式按最终文案，混剪可切换固定素材模式或音频模式，去重按导入视频。
- 音频来源：任务音频队列、混剪音频队列或服务默认语音；混剪音频只作为音轨/时长依据，不进入画面素材池。
- 字幕：一个开关同时控制预览、字幕文件生成、成片合成、HeyGen 字幕下载和 ASR 兜底。
- 输出内容：用户在输出前能看到将导出 MP4、封面、字幕、剪辑记录或发布资料包中的哪些文件。

字幕开关的语义必须是端到端的。关闭字幕后，数字人链路不保存 HeyGen 返回的字幕，也不触发 ASR 兜底；混剪链路不生成 SRT，也不把字幕复制进导出包。

## Consequences

- 新手只需要先看统一设置，再补每个模式专属素材。
- UI 可以继续保留高级模式专属设置，但不能让用户为了基础输出到处找开关。
- 后续如果加入外部音频直传 HeyGen 或其它视频模型，也应先接入统一生成设置，再扩展模式内部细节。
