# 保真去重处理设计

## 目标

视频去重处理模式面向自有素材、授权素材和本软件生成的混剪/成片，目标是降低内部重复风险评分，同时尽量保持成片观感。软件的评分不代表平台官方判定，也不承诺任何平台结果。

## 策略

新增三档策略：

1. 保真轻去重：轻微裁切、轻微色彩扰动、轻微变速、重新编码、重做字幕/封面。
2. 保真强去重：默认策略，在肉眼观感接近的前提下增加重采样、帧率变化、噪声、锐化、GOP/编码结构变化。
3. 深度像素重塑：用于高风险素材，增加更强的像素扰动、重采样和编码变化，仍尽量保持主画面可识别。

旧策略 `light-polish` 映射为保真轻去重，旧策略 `content-rewrite` 映射为保真强去重。

## 评分

继续输出 0-100 的内部原创度/重复风险评分，默认 80+ 才通过。评分维度包括：

- 片段/帧结构变化
- 原素材复用风险
- 视觉扰动强度
- 字幕、标题、封面变化
- 音频轨道变化
- 文案相似风险
- 水印风险

评分报告必须说明：该分数是软件内部可解释评分，不代表平台官方判定。

## UI

去重模式的“处理策略”下拉框显示三档新策略，并把“保真强去重”作为默认推荐。提示文案使用“重复风险降低/内部评分”，不使用“保证过审”或类似承诺。

## 验收

- 本地质量门禁通过：typecheck、lint、test、build、format。
- 去重单测覆盖三档策略。
- 真实去重 E2E 至少跑通一条 MP4，输出处理后 MP4、字幕、封面和评分报告。
